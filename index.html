Como calculamos las rutas
  • Tiempo base = suma de tiempo_min de cada tramo.
  • Por cada condicion activa: tiempo = tiempo x (1 + tiempo_pct / 100).
  • Costo total = suma de costo + suma de costo_extra de condiciones.
  • Orden por defecto: tiempo. Tambien puedes ordenar por costo o transbordos.
